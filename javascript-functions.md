# Javascript Functions Notları

### Fonksiyon tanımlama
İki türlü de tanımlanabilir. Eğer fonksiyon içinde tek bir işlem varsa 

```javascript
function App () {
    someLogic
    return object;
}

function App (sum) => sum * 2 ;
function App () => a + b ;

const App = () => {
    someLogic
    return object ;
}
```
